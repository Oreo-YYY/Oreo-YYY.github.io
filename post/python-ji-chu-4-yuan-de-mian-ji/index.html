Python挑战1-圆的面积

2020-08-15 1 min read

目标

  • 创建一个 Python 脚本 CircleArea.py
cd /home/shiyanlou/Code
touch CircleArea.py
  • CircleArea.py 能够计算出一个半径为 2 的圆的面积，并且把面积打印出来，保留小数点后 10 位
  • 不要使用 input 等方法获得输入，程序不需要输入任何参数，程序执行如下，直接输出半径为 2 的圆的面积

提示语

  • import math
  • 使用 format 可以控制输出的小数位数，可以使用 '{:.10f}'.format() 来进行格式化输出保留 10 位小数
  • 请不要使用 input 获取输入，程序执行不需要任何输入的参数
#!usr/bin/env python3
import math
# 计算圆的面积
area = 2 * 2 * math.pi
# 格式化输出圆的面积，保留10位小数
print("{:.10f}".format(area))
